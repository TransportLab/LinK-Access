# Link-Access
Evaluating proposed links on accessibility


Generating distance-based catchment areas (isochrones)

This Python code is a processing tool for QGIS that evaluates the accessibility of potential links on a specific network. The methodology is stated in Lahoorpoor and Levinson (2020)

Reference: Lahoorpoor, B., & Levinson, D. M. (2020). Catchment if you can: The effect of station entrance and exit locations on accessibility. Journal of Transport Geography, 82, 102556.
